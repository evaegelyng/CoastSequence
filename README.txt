Code, environmental data and laboratory metadata for the paper "Consistent drivers of coastal biodiversity across the tree of life".

Summary of the study: Coastal ecosystems are threatened by global change but remain understudied despite their importance for biodiversity and human societies. For instance, most coastal studies and monitoring programs focus on a few groups of organisms, due to methodological limitations. However, high-throughput sequencing of DNA from environmental samples (eDNA) now facilitates taxonomically broad studies of coastal biodiversity. Here, we conducted a very comprehensive study of coastal biodiversity patterns using multi-marker eDNA metabarcoding of nearly 1,000 water and sediment samples covering the strong salinity gradient from the North Sea to the Baltic Sea. We included the major coastal habitat types, and measured a range of environmental variables. 

File structure: The repository contains four directories; "environmental_data" which contains all the environmental measurements, "lab_metadata" which contains the metadata regarding laboratory processing of the samples, "HPC" which contains the code run on a high performance computing cluster, and "local" which contains the code run locally on a laptop computer. The latter two directories also contain necessary input files, i.e. files that are not generated by the scripts, with the exception of fastq files and related files, which are available on Dryad at https://doi.org/10.5061/dryad.gmsbcc334.
