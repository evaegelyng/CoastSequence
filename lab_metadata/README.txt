This folder contains two files needed for filtering the eDNA sequences based on negative controls:

1. extraction_refs_both_seasons.txt
    - Description: Indicates for each sample ("name"), the ID of the extraction batch it belongs to ("extraction_number").
2. PSU_refs_both.txt
    - Description: Indicates for each sample ("name), the ID of the PSU (PCR Setup Unit, "PSU") it belongs to, as well as the
    sampling season ("season") and whether the sample is a water of sediment sample ("substrate_type").
